# ASCII Art 重設計：祈福儀式 & 驅魔儀式

Date: 2026-05-12

## 背景

`kismet/agent/tools/renderer.py` 中的 `show_blessing()` 和 `show_exorcism()` 目前使用簡單的 ASCII art（5–8 行，樣式粗糙）。需要重設計為更有儀式感的版本。

## 設計決策

| 項目 | 決定 |
|------|------|
| 風格 | 道教法陣風 |
| 尺寸 | 中等（約 15 行） |
| 版型 | 符籙卷軸式，內嵌八卦列 |
| 祈福色調 | 外框金色、八卦列紫色、內文金色 |
| 驅魔色調 | 外框血紅、八卦列血紅（深）、內文血紅 |
| 兩者區分方式 | 同結構，用顏色和文字區分吉/凶 |

## 實作範圍

只修改 `kismet/agent/tools/renderer.py`：

- `show_blessing()`（line 524）：替換 `candle_art` 變數內容
- `show_exorcism()`（line 548）：替換 `art` 變數內容

不動其他任何東西（Panel 標題、stats 文字、呼叫端邏輯不變）。

## 注意事項

- Chinese 字符在 terminal 佔兩個 column，實作時需精確校正每行寬度使外框對齊
- Rich `Text.from_markup()` 的色彩標記需分段套用
- `Align.center()` 會處理置中，不需手動加空格
